# DuckyServers Assets

All of the configs, plugins, maps, and other assets powering DuckyServers.

## Usage

1. Download [GSOchestrator](https://github.com/pongo1231/GameServerOrchestrator), extract it somewhere and follow the README
2. Clone this repo into a tf subdir
3. Create an updater/ directory with steamcmd in it
4. Run update.sh or copy your own tf2 srcds installation to common/00-base
5. Adjust the individual start.sh and other files to your taste
6. Start the servers using GSOrchestrator and enjoy!
